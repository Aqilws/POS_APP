# Install-Package
Install package for running this project

# flask
* npm
  ```sh
  pip install flask
  ```

# requests
* pip
  ```sh
  pip install requests
  ```

# pillow
* pip
  ```sh
  pip install pillow
  ```

# customtkinter
* pip
  ```sh
  pip install customtkinter
  ```

# Running Project 

* Api.py
  ```sh
  python api.py
  ```
* app_gui.py
  ```sh
  python app_gui.py
  ```
